# Last-Pieces-Moved-In-Chess-Project

This repository contains a data analysis and machine learning project originally developed for the UC Irvine Math 10 course (Winter 2025). Using a dataset of chess games from LiChess, this project investigates how different factors impact game outcomes. The analysis specifically focuses on visualizing win rates based on player ratings and utilizing machine learning algorithms (Decision Tree and K-Nearest Neighbors) to predict which piece was moved last in a game based on total turns, game length, and total captures.
